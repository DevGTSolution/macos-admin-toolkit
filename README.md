# macOS Admin Toolkit

A collection of macOS scripts, Jamf Extension Attributes, and Apple endpoint automation examples for real-world device management workflows.

This repository is focused on practical Apple endpoint engineering patterns, including inventory, compliance checks, troubleshooting helpers, application deployment workflows, and user-friendly automation concepts.

## Contents

### Scripts

- `remove-pages-numbers-keynote.sh`  
  Removes Apple iWork applications when they are installed and not currently running.

- `installomator-swiftdialog-self-service-apps.sh`  
  Example workflow for using Installomator and SwiftDialog to support Self Service application installs.

### Jamf Extension Attributes

- `ea-crowdstrike-status.sh`  
  Reports CrowdStrike Falcon status for Jamf inventory.

- `ea-uptime-status.sh`  
  Reports device uptime for inventory and reporting.

- `ea-admin-users-status.sh`  
  Reports users with local admin rights.

## Focus Areas

- macOS endpoint management
- Jamf Pro workflows
- Extension Attributes
- device compliance checks
- patching and update workflows
- SwiftDialog-style user prompts
- automation for Apple IT teams
